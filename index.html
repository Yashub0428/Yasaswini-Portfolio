Publication

LAN Network Stability Prediction using Hybrid ARIMA-LSTM Framework
Accepted at IEEE ICCCNT 2025, IIT Indore

Agenda: The goal of this research is to improve the prediction of LAN network instability—particularly in detecting sudden latency spikes and disruptions due to congestion or hardware anomalies. It aims to support proactive network management and maintain consistent Quality of Service (QoS).

Methodology: The model combines ARIMA (AutoRegressive Integrated Moving Average) to capture linear trends with LSTM (Long Short-Term Memory) to model nonlinear, time-dependent behavior. This hybrid architecture leverages the strengths of both statistical and neural network methods to better analyze time-series latency data.

Novelty: Unlike previous approaches, this work integrates Fast Fourier Transform (FFT)-based feature engineering, rolling statistics, and trend slopes to enhance predictive accuracy. The combination of ARIMA’s interpretability and LSTM’s deep learning capabilities allows the model to learn both steady and volatile patterns in latency.

Outcome: The proposed hybrid model demonstrated superior accuracy and stability compared to standalone ARIMA or LSTM models. It effectively anticipates instability in real-time LAN environments, making it suitable for smart network monitoring systems.
